Quantum Tutor Manual Operativo

Version operativa actual: v6.1-stateless

Este manual describe el runtime vigente del proyecto. Los documentos v1.x, v2.0, v4.x y v5.1 se conservan solo como contexto historico.

Componentes principales

  • quantum_tutor_orchestrator.py: pipeline stateless por request.
  • rag_engine.py: ingesta, recuperacion y referencias de imagen.
  • api_quantum_tutor.py: API FastAPI.
  • app_quantum_tutor.py: interfaz Streamlit.
  • multimodal_vision_parser.py: analisis de imagenes y derivaciones.
  • tool_scheduler.py: decision sobre RAG y Wolfram.

Contrato actual

  1. Se crea un QuantumRequestContext por consulta.
  2. El orquestador detecta tema, intencion y plan de herramientas.
  3. RAG y Wolfram pueden ejecutarse en paralelo.
  4. Si el cliente LLM no esta disponible, se responde con fallback local.
  5. La metadata final reporta latencia, imagenes, tema y uso de Wolfram.

Comandos utiles

streamlit run app_quantum_tutor.py
uvicorn api_quantum_tutor:app --reload
python -m pytest -q
