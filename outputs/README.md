# Outputs

This directory contains generated artifacts that are still part of the current project surface.

- `qst_results.json`: latest stored Quantum Stress Test report.
- `evaluation_dashboard.html`: dashboard artifact associated with the QST report.
- `system_status.json`: status snapshot for the tutor runtime.
- `checkpoints/`: resumable OCR and ingestion state that should stay outside the repository root.
- `reports/`: secondary simulation outputs.

These files are outputs, not source-of-truth documentation.
